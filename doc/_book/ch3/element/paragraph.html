段落组件

段落组件

段落组件用来封装文本内容、图片或其他行间 DOM 元素或行间组件。

作用

段落组件为块级组件，表示一个片段分块。内部可嵌套行间 DOM 元素或行间组件，常用来作为一段文字描述的容器。

语法

调用方式
import Paragraph from '@Paragraph';
<Paragraph>
    This is a text.
</Paragraph>
渲染结果
<p>
    This is a text.
</p>
默认样式

最佳实践

通过 公共配置，组件可以实现丰富的样式和逻辑的控制。

导言

Omega 预留了一种针对文章导言的 class 修饰符。

修饰符名 描述
