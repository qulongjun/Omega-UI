# _includeClass
> 为组件追加 Class 修饰符，指定该组件最外层 DOM 元素增加指定的 Class 修饰符。

### 作用
该属性可以为组件增加
 
### 语法
``` html
<style type='text/css'>
.m-font-blod:{
    font-weight:blod;
}
</style>
<Text _includeClass='m-font-blod'>This is a text. </Text>
```

### 渲染效果
``` html
<span class='m-font-blod'>
    This is a text.
</span>
```
