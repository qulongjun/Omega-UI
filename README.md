## 快速构建

推荐使用快速构建工具  `Omegaui-CLI`  搭建项目，只需依次执行：

``` #Bash
// 安装快速构建工具
$ npm install omegaui-cli -g

//初始化项目
$ omegaui-cli init my-project
```

> 构建工具具体用法清参考[构建工具说明](https://github.com/qulongjun/Omega-cli)。
